General Addresses
US Address JP Address Icon Length Description Notes Default Value Normal Value Range
0608D4 05D0A4 1 bytes Debug mode. Change this value to enable or disable debug mode. 1 1 - 0 (OFF/ON, yes it's backwards.)
12B74C - 2 bytes Change Current Room. Set a specific room number to this address. After the value is set, you must change the teleport activator to 1. Current room Value. The exact number of rooms in the game is unknown.
12B752 - 1 byte Room teleport Activator. Activates teleportation to the designated room in Current Room. 0 1 - 0 (ON/OFF)
060829 - 1 byte Debug Jump. Enable Debug Jump, it allows flint to make multiple jumps in the air. 0 1 - 0 (ON/OFF)
0FA0F4 - 1 byte Hitbox Viewer. Change this value to see or unsee enemys, objects and characters hitbox. 0 1 - 0 (ON/OFF)
0FA0C7 - 1 byte Initial Gameplay state. (Useful if you want to test the game and doesn't want to wait). 1: Start from tutorial, 2: Select from memory card, 3: Main menu, 4: Skip tutorial, start from airship Storage room. 0 0 - 4
0FA0C4 - 1 byte Open game menus. 1: Item Shop, 2: Nothing, 3: Memory Card Save Screen, 4: Memory Card Load Screen, 5: Opens Flint's pocket. 0 0 - 5
Player Addresses
US Address JP Address Icon Length Description Notes Default Value Normal Value Range
067712 - 2 bytes Flint's Life. (Values above the limiter cause the game to crash. see Flint's Life limiter). 200 0 - 999
067710 - 2 bytes Flint's Life limiter. Change this value to increase or decrease the limiter. 200 0 - 999
11D631 or 11D641 - 1 byte Flint's Speed. 0: Stopped, 8: Walking, 9: Walk+Jump, 14:Running, 17: Run+Jump. 0 0 - 17
067704 - 4 bytes Flint's Gold. Change Flint's gold, 0x12F59C is in control of animation. 100 0 - 999999
06744F - 1 byte Flint's Key Amount. Change Flint's Key amount. 0 0 - 9
067425 - 1 byte Enable EP. Enables Flint's EP bar. 0 1 - 0 (ON/OFF)
067716 - 2 bytes Flint's EP. Values above the EP limiter cause the game to crash. see Flint's EP limiter. 200 0 - 999
067714 - 2 bytes Flint's EP limiter. Change this value to increase or decrease the limiter. 200 0 - 999
06746C - 1 byte Flint's Weapon 31 - Long Sword, 32 - Bronze Sword, 33 - Broad Sword, 34 - Rune Sword, 35 - Shining Sword, 36 - Rising Sword. 31 [31, 32, 33, 34, 35, 36] (Anything else is another item and may crash the game).
06746E - 1 byte Flint's Shield 39 - Buckler Shield, 40 - Small Shield, 41 - Titan Shield, 42 - Mithril Shield, 43 - Adamant Shield, 44 - Valar Shield. 39 [39, 40, 41, 42, 43, 44] (Anything else is another item and may crash the game).
067464 - 1 byte Flint's Pocket 1 This Value Holds the pocket 1 item, change it to have any item on it. 255 (23 - 26), (51 - 58), (70 - 74), (76 - 77)
067465 - 1 byte Flint's Pocket 2 This Value Holds the pocket 2 item, change it to have any item on it. 255 (23 - 26), (51 - 58), (70 - 74), (76 - 77)
067466 - 1 byte Flint's Pocket 3 This Value Holds the pocket 3 item, change it to have any item on it. 255 (23 - 26), (51 - 58), (70 - 74), (76 - 77)
Map Addresses
US Address JP Address Icon Length Description Notes Default Value Normal Value Range
- - 1 byte Particles state flag. Controls the state of particles loaded on the map. 0 1 - 0 (ON/OFF)
0FA0C6 - 1 byte Room Visibility. See or unsee elements of the current room. 0: View all textures and objects, 1: View only objects, 2: View only texture, 3: View nothing. 0 0 - 3
